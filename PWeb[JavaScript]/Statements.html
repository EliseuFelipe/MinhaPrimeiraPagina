JavaScript Instruções

Exemplo:
var x, y, z; // instruções 1
x = 5; // instruções 2
y = 6; // instruções 3
z = x + y; // instruções 4
Programas JavaScript
Um programa de computador é uma lista de "instruções" a serem "executadas" por um computador.
Em uma linguagem de programação, essas instruções de programação são chamadas de instruções.
Um programa JavaScript é uma lista de instruções de programação.
Em HTML, os programas JavaScript são executados pelo navegador da web.
Declarações JavaScript
As instruções JavaScript são compostas por:
Valores, Operadores, Expressões, Palavras-chave e Comentários.
Esta declaração diz ao navegador para escrever "Hello Dolly". dentro de um elemento HTML com id = "demo":

Exemplo


document.getElementById("demo").innerHTML = "Hello Dolly.";
Keyword Description
break termina uma repetição
continue volta para o começo da repetição
debugger Stops the execution of JavaScript, and calls (if available) the debugging function
do ... while Executa um bloco de instruções e repete o bloco, enquanto uma condição é verdade
for Marca um bloco de instruções a serem executadas, contanto que uma condição seja verdadeira
function Declares a function
if ... else Marca um bloco de instruções a ser executado, dependendo de uma condição
return sai da função
switch Marca um bloco de instruções a serem executadas, dependendo dos diferentes casos
try ... catch Implementa o tratamento de erros em um bloco de instruções
var Declara uma variavel
