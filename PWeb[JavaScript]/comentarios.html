JavaScript Comentários
Comentários JavaScript podem ser usados para explicar o código JavaScript e torná-lo mais legível.
Comentários JavaScript também podem ser usados para impedir a execução, ao testar códigos alternativos.
Comentários de linha única
Comentários de linha única começam com //.
Qualquer texto entre // e o final da linha será ignorado pelo JavaScript (não será executado).
Este exemplo usa um comentário de linha única antes de cada linha de código:

Exemplo


// Change heading:
document.getElementById("myH").innerHTML = "My First Page";
// Change paragraph:
document.getElementById("myP").innerHTML = "My first paragraph.";
Este exemplo usa um comentário de linha única no final de cada linha para explicar o código:

Exemplo


var x = 5; // Declare x, dê o valor de 5
var y = x + 2; // Declare y, dê o valor de x + 2
